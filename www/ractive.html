HACE - Ractive

Description

This page provides you with a demonstration of ractive.js and how to code this example. Type a number into the input field and watch the size of the text change.


Code

<script id="myTemplate" type="ractive">
<p style="font-size: {{content}}px;">
This is a double mustache: <u>"{{content}}"</u> it displays what is in content.<br/>
This is a triple mustache: <u>"{{{content}}}"</u> it executes code in content.
</p>
<p><label>Enter text size:<input value="{{content}}">px</label></p>
</script>
<p id="myElement"></p>
<script>
var ractive = new Ractive({
el: '#myElement',
template: '#myTemplate',
data: {
content: '17'
}
});
</script>
